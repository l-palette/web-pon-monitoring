О сайте

Раз в минуту происходит заполнение нереляционной базы данных логами - уровнями по устройствам. У каждого устройства клиента есть лицевой счет и по нему уникально идентифицируется устройство. По запросу пользователя данные об уровнях запрашиваются из базы данных и выводятся на экран в виде графиков.
